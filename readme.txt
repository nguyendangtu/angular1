This is a simple site of starhub. To run this example, we can follow step by step as below:
1. run maven build by using command - mvn clean install
2. copy ROOT.war and parse to context root of web server or application server.
3. we can go to person page by path /personal or /personal/amin
4. we can go to admin page by path /admin**, username and password login is admin/admin
